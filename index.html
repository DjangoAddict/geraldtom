Flask / Raw Postgresql
Real Estate Investment Advisory has lead me to an interest in real estate data. This is an analytical warehouse of New York building and ownership data comprised of billions of cells of data using data engineering to optimize the database allowing queries to be processed in milliseconds. Data was aggregated from various sources. An ERD diagram and bridge tables were then established for the architecture. Optimization techniques were then applied in raw PostgreSQL (SQL-Language) using Psycopg2 (Database Driver) to interface with Flask (Python Framework) as the frontend user-interface. The program is being used as a proprietary database to generate client leads.
